# historia-windows-pesquisa
# 🪟 Evolução do Sistema Operacional Windows

Repositório destinado à pesquisa histórica das versões do Microsoft Windows, realizado para a disciplina de [Disenvolvimento de Sistemas].

## 📜 Linha do Tempo

| Versão              | Ano       | Principal Novidade / Característica marcante                          | Curiosidade / fato interessante                                                                 |
|---------------------|-----------|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| **Windows 1.0**     | 1985      | Interface gráfica sobre o MS-DOS                                      | O nome original do projeto era "Interface Manager".                                             |
| **Windows 2.0**     | 1987      | Janelas sobrepostas, ícones e suporte a atalhos de teclado           | Foi processada pela Apple por suposta cópia da interface do Macintosh.                           |
| **Windows 3.x**     | 1990-1992 | Gerenciador de Programas, modo protegido, suporte a 256 cores         | Foi a versão que popularizou o jogo *Paciência* para ensinar o uso do mouse.                    |
| **Windows 95**      | 1995      | Menu Iniciar e Barra de Tarefas                                       | A Microsoft pagou milhões para usar a música "Start Me Up" dos Rolling Stones no comercial.     |
| **Windows 98**      | 1998      | Internet Explorer integrado, suporte USB nativo, FAT32                | A edição Second Edition (98 SE) trouxe compartilhamento de internet (ICS).                      |
| **Windows ME**      | 2000      | Restauração do Sistema, Movie Maker, interface mais colorida         | Considerado por muitos como o Windows mais instável da história.                                |
| **Windows 2000**    | 2000      | Kernel NT profissional, Active Directory, suporte a múltiplos CPUs   | Muito usado em empresas; serviu de base para o XP.                                              |
| **Windows XP**      | 2001      | Estabilidade (Kernel NT) e novo visual "Luna"                         | A famosa imagem do papel de parede (Bliss) é uma foto real, sem Photoshop.                      |
| **Windows Vista**   | 2007      | Interface Aero (transparência), UAC, busca indexada                   | Exigia hardware superior ao comum na época → muita rejeição inicial.                            |
| **Windows 7**       | 2009      | Refinamento do Aero, bibliotecas, barra de tarefas aprimorada        | Considerado por muitos o melhor Windows da história.                                            |
| **Windows 8**       | 2012      | Interface Metro (tiles), tela inicial otimizada para toque            | Removeu o Menu Iniciar clássico → gerou grande rejeição de usuários.                            |
| **Windows 10**      | 2015      | Retorno do Menu Iniciar completo, Edge, atualizações contínuas        | Anunciado como "o último Windows" (não foi); modelo "Windows as a Service".                     |
| **Windows 11**      | 2021      | Menu Iniciar centralizado e suporte a apps Android                    | É a primeira versão que exige obrigatoriamente o chip de segurança TPM 2.0.                     |

> **Nota:** Todas as versões intermediárias solicitadas foram adicionadas (Windows 2.0, 98, ME, 2000, Vista, 7, 8, 10). Incluí também Windows 3.x e 8 (com menção implícita a 8.1 como atualização) para manter a linha cronológica completa e fluida.
